MinkTilet Architecture & Tech Stack

A lightweight, high-performance web app deployable to Vercel. Open http://localhost:52490 to view.

SYSTEM ARCHITECTURE
📱 Client (Next.js / React) • Responsive Design: CSS grid/flexbox adapting instantly from mobile (single-column card view) to desktop (three-panel console).
• Media Capture: HTML5 Web Audio API (voice recording) + canvas frame extraction (webcam posture check).
• State Machine: Client-side state storing Reasoning Logs in real-time.
⚡ Vercel Serverless API • Google AI Studio Bridge: Standard Serverless Functions invoking the free-tier SDK.
• Grounding Core: Ingests prompts and appends local Ethiopian context (fasting schedules, traditional herbal mappings, local air quality).
• Mock Integrations: Fast local mock endpoints simulating WeVa database, Kuriftu bookings, and Telebirr checkout.
✨ Key Model Choice: Gemini 1.5 Flash (Google AI Studio Free Tier)

While Gemma 2 (9B/27B) is excellent for pure text, Gemini 1.5 Flash is the ultimate hackathon hack. Because it is natively multimodal, it allows our agent to directly ingest PDFs (bloodwork), audio files (voice notes), and photos (skincare ingredients) on the free tier with 0ms pre-processing overhead.
